# Photoshop Layer Guide

Document size: 1774 × 887 px. Work at 2× for exported assets when possible.

## Layer Groups

FRAME
  - Black metal base: Normal 100%, bevel 2–5 px, subtle noise 2%.
  - Chrome bevels: Screen/Overlay highlights, black inner shadows.
  - Mesh grilles: Multiply 80%, clipped inside vent masks.

LEFT PANEL
  - Red brushed aluminum base: #8B1119, horizontal motion blur brushing.
  - Edge darkening: Multiply 25%.
  - Inner glow red: Linear Dodge 15%.
  - Bolts: Normal 100%, bevel and shadow.

RIGHT PANEL
  - Same as LEFT PANEL.

CENTER SCREEN
  - Bezel chrome: Gradient Overlay, Bevel & Emboss.
  - Glass: #070707 Normal 100%, diagonal reflection overlay Screen 8–15%.
  - UI graphics: Red #FF2A2A, text #E0E0E4.

KNOBS
  - Shadow
  - Outer black ring
  - Rubber grip ring
  - Metal face radial gradient
  - Pointer red/white
  - Tick marks
  - Reflection
  - Hover/pressed glow overlays

SLIDERS
  - Track groove
  - Red LED fill
  - Handle base
  - Chrome highlight
  - Shadow

BUTTONS / SWITCHES / LED
  - Normal
  - Hover glow
  - Pressed inner shadow
  - Selected red fill
  - Disabled 40% opacity

TEXT
  - Logo
  - Labels
  - Values
  - Screen labels
  - Button labels

OVERLAYS
  - Vignette
  - Dust/scratches
  - Red reflection blooms

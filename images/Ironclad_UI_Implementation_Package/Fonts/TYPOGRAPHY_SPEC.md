# Typography Specification

Logo: wide extended techno/sci-fi font, 42–54 px at source scale, chrome/red bevel treatment.
Labels: uppercase, 18–22 px at source scale, tracking +40, color #E0E0E4.
Values: 18–24 px, color #FF2A2A.
Large display number: 80–110 px, bold, color #FF2A2A, subtle outer glow.
Preset title: 28–34 px, uppercase, color #FF2A2A.
Screen labels: 15–19 px, uppercase, muted #A8A8A8.
Button labels: 14–18 px, uppercase, centered.
Baseline alignment: keep value text centered under knobs and sliders; labels centered above controls.
